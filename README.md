# Eat-and-Go
>### This project is written in Java.
>### 創作背景
>
>此主題希望可以讓使用者走到哪裡都可以知道當地的餐廳資訊，因此決定先以學校附近的早餐店、早午餐、餐廳等為範圍。
>### 創意性 
>只要走到附近，就可以顯示附近可用餐的地方，並附上菜單價位及營業時間。大大減少用餐前搜尋的時間，並可以在用餐前先估算是否合乎自己的預算。
>### 未來性
>使用的人增加，則參與的餐廳也會越多，越多人使用，資訊越完整。未來更可以透過垂直整合來增加上下游的合作關係，達到減少成本的目標。透過資料庫可將訂單資訊傳給各個餐廳，也可以結合定位功能，讓消費者不用擔心到現場需要候位，也可讓定位的功能簡單化！
>### 可行性
>在設計上，需要考慮排版、多媒體的顯示、點擊確認、訂單查詢及總價錢計算。
